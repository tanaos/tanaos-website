How it works

Integrate the moderation model we've created for you in your pipeline with just a few lines of code, using the Transformers library.

from transformers import pipeline

# Load the custom chatbot moderation model
pipeline(
    task="text-classification",
    model="your-org/chatbot-safety-moderation",
)

responses = [
    "I'm sorry, I can't help with that.",
    "Here's how to make a bomb...",
    "Let's talk about something else.",
    "You are stupid!",
]

results = pipeline(responses)
